XCode 无法自动补全

发表于 2016-05-07   |   分类于 iOS开发   |     |   阅读次数

吐槽

NNGD,写着写着代码又无法自动补全了，难道要我默写吗？做iOS的肯定都遇到过无法自动补全的问题，这里写出我遇到的一些BUG与解决办法。

原因

1.XCode自带BUG

有些时候我们没有调用第三方库，发现UIkit的自动补全都没法触发了，这样的原因多半是因为XCode自带BUG，在你的注释里有中文的时候，不会触发自动补全，比如这个’年’，删掉过后再编译一下试试，再不行就重启一下XCode。

2.要import pods里的头文件

有时候XCode搜索头文件不会去搜索Pods里的第三方库，我们需要在XCode里设置一下项目Target的Search Paths。

如图，双击User Header Search Paths的空白处，填入”$(PODS_ROOT)”(没有引号), 设置为递归的搜索recursive，问题解决了。
